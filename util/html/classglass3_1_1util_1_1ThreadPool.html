Detailed Description

util threadpool class

This class supports creating, starting, stopping, and monitoring a pool of work threads, managing a FIFO queue for jobs to be processed, and allowing for specific pool name and sleep between work time.

bool glass3::util::ThreadPool::healthCheck ( )

pool health check function

Checks to see if each thread that runs the jobLoop() function is still operational, by checking the value of m_ThreadStatusMap[id] (==true) every m_iHealthCheckInterval seconds, setting m_ThreadStatusMap[id] to false after the check. It is expected that the jobs processed by the thread pool will finish within m_iHealthCheckInterval seconds

Returns
returns true if the pool is still running after m_iHealthCheckInterval seconds, false otherwise
