Detailed Description

neic-glass3 configuration class using JSON formatting

The glass Config class is a class used to read JSON formatted configuration files from disk. The Config class filters out comment lines (signified by '#'), newlines, and white space, and provides the configuration as a JSON object.

Warning
This class is NOT thread safe
