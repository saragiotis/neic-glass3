Detailed Description

glass fileOutput class

The glass fileOutput class is a thread class encapsulating the detection fileOutput logic. The fileOutput class handles fileOutput messages from from glasscore, and writes the messages out to disk.

fileOutput inherits from the threadbaseclass class. fileOutput implements the ifileOutput interface.
