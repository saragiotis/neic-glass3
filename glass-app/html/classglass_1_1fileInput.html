Detailed Description

glass fileInput class

The glass fileInput class is a thread class encapsulating the data fileInput logic The fileInput class handles reading fileInput data from disk, parsing it, validating it, and queuing it for later use by the associator class

fileInput inherits from the glass3::input class. fileInput implements the iinput interface.
