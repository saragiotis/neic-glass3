Detailed Description

glass association class

The glass association class is a thread class encapsulating the glass core association and nucleation engine. The associator class pulls input data from the input class, and sends it into glasscore. It also routes any results to the output class, and sends station information requests to the stationlist class. The class also sends any configuration into glasscore.

associator inherits from the threadbaseclass class.

associator implements the IGlassSend and iassociator interfaces.
